# Formula Intelligence — Analisis de Ingredientes Coca-Cola Zero Sugar

Proyecto de analisis de datos aplicado a la gestion de ingredientes en la industria de bebidas.

## Contenido
- `proyecto_ingredientes_cocazero.ipynb` — Notebook con analisis completo de ingredientes,
  estatus de aprobacion regulatoria y dashboard de visualizacion.

## Herramientas
- Python (pandas, numpy, matplotlib)
- Normativas aplicadas: NOM-218, NOM-251, Codex Stan 192, FDA 21 CFR

## Contexto
Ejercicio desarrollado como parte de mi preparacion en Formula Intelligence,
gestion de bases de datos de ingredientes y cumplimiento normativo alimentario.
