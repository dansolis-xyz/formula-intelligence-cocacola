# Formula Intelligence — Análisis de Ingredientes

Proyecto de análisis de datos aplicado a la gestión de ingredientes en la industria de bebidas.

## Contenido

- `proyecto_ingredientes_cocazero.ipynb` — Notebook en Python con análisis completo de ingredientes de Coca Cola Zero Azúcar, estatus de aprobación regulatoria y dashboard de visualización.
- `analisis_usos_tecnicos_FDA.R` — Script en R con exploración, limpieza y visualización de la base de datos pública FDA Substances Added to Food (EAFUS), que contiene más de 3,900 ingredientes aprobados clasificados por uso técnico.
- `analisis_usos_tecnicos_FDA.png` — Gráfica generada: Top 10 usos técnicos de sustancias aprobadas por la FDA.

## Herramientas

- Python (pandas, numpy, matplotlib)
- R (ggplot2, dplyr, stringr)
- Normativas aplicadas: NOM-218, NOM-251, Codex Stan 192, FDA 21 CFR

## Contexto

Ejercicio desarrollado como parte de una simulación como Formula Intelligence Scientist, gestión de bases de datos de ingredientes y cumplimiento normativo alimentario.
